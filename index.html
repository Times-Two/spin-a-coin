Privacy Notice & Important Information

Important Notice to All Users

Before using the Cafe Lucky Wheel, please read and understand the following important information:

Critical Warning:

Do not refresh the browser while using this application. Refreshing the page will cause:

  • Loss of all current game progress and unsaved data
  • Termination of your active session
  • Forfeiture of any in-progress bets or winnings

Data Loss Policy

Please note that in the event of a page refresh or browser crash:

  • No refunds will be provided for any lost data, bets, or winnings
  • Game sessions cannot be recovered
  • Account balance will reflect only the last successfully saved state

Problem Reporting

If you encounter any issues:

  • Please report immediately to cafe staff
  • Do not attempt to refresh the page or restart the application yourself
  • Staff will assist with any technical problems

Session Management

For your security and data integrity:

  • Your game session is valid for 30 minutes
  • After expiration, you'll need to pay 2 GHS to continue playing
  • Always logout properly when finished playing
